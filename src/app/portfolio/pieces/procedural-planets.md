# Overview 

This pair planet, which was created as a test for the procedural textures. The textures were created using Blender's procedural texture system, which allows for a high level of detail and realism without the need for UV mapping or image textures.

[CAROUSEL: /Media/IcePlanetCompressed.png, /Media/DesertPlanet.png]

## Process

### Base Model

Despite its appearance, the planet is actually a simple sphere with a procedural material applied to it. The material uses a combination of noise textures and color ramps, to create the black and white map. Different shades of the map are then linked to colors, and thats it.

### Athmosphere

Another sphere with a transparent material and an emission shader was used to create the atmosphere. The material uses a color ramp to create a gradient effect that gives the appearance of an atmosphere, but no, there is not athmocsphere, just a sphere with a material that gives the illusion of an atmosphere.
